# ArkSigner Manager

ArkSigner Manager is a GTK4-based GUI for running ArkSigner reliably via a Debian container (systemd-nspawn) on Arch and Fedora.

## Features
- GTK4 + Python GUI
- Container-based execution for stability
- Packaging for Arch (PKGBUILD) and Fedora/RPM
